## Projects

### [apply-pilot](https://github.com/shashikirandevadiga/apply-pilot)
**Problem:** Job applications took 45 minutes each. I was applying to 20+ roles a week — 15+ hours of manual work.
**What it does:** 8-agent Claude Code pipeline that generates a tailored resume, cover letter, and outreach strategy from a job description. 45 min → 5 min per application.
**Demonstrates:** Agent orchestration, multi-step pipeline design, quality verification loops, human-in-the-loop checkpoints.
**Stack:** Claude Code · Python · multi-agent orchestration
**4 ⭐ · 5 forks**

### [claude-statusline](https://github.com/shashikirandevadiga/claude-statusline)
**Problem:** Working in Claude Code for hours with zero visibility into context usage, rate limits, or session cost. No situational awareness.
**What it does:** Extended Claude Code statusline with 11 features — color-coded context bar, 5-hour rate limit countdown, YOLO mode warning, cache hit ratio, spend vs. monthly limit.
**Demonstrates:** Product thinking applied to developer tooling — identified a real gap, extended existing OSS, shipped specific features for specific problems.
**Stack:** Bash · Claude Code API
**Built on:** [kamranahmedse/claude-statusline](https://github.com/kamranahmedse/claude-statusline)
